# La fonction `print()`

La fonction `print()` est une fonction intégrée de Python qui permet d'afficher des messages à l'écran. 

```python
print("Hello, World!")
```

``` 
Output: Hello, World!
```

On peut afficher plusieurs messages en même temps en les séparant par une virgule.

```python
print("Hello", "World!")
```

```
Output: Hello World!
```
