Un logiciel est un programme permettant l'exécution d'un ensemble de tâches prédéfinies. Il se
"greffe" au système d'exploitation afin d'offrir à l'utilisateur des fonctionnalités complémentaires et
personnalisées.
<br />
Un logiciel peut être une banale application (anti-virus, outils bureautique, messagerie, navigateur
web…), mais aussi une menace (virus, chevaux de Troie, cryptolockers…).
<br />
**Les types d'installations**:
- **L'executable (.exe)**: C'est le type d'installation le plus courant. Il suffit de lancer le fichier
d'installation et de suivre les instructions.
- **Le zip auto-extractible (.zip)** : Le programme est dans un dossier compressé, il se
décompressera et s'installera automatique
- **Le Microsoft Self Installer (.msi)** : C'est un fichier d'installation Microsoft. Il est souvent
utilisé pour les mises à jour de Windows et les logiciels Microsoft. Il suffit de lancer le fichier
d'installation et de suivre les instructions.
<br />
**Les sources d'installation**:
- Les supports optiques, les stockages électroniques (HDD, SSD, SD, USB), le réseau local, le
cloud, et par extension, l'Internet.
<br />

## Avant l'installation

- Vérifier la version (compatibilité logiciel/système d'exploitation)
- Vérifier l'architecture x86 ou x64
- Vérifier le minimum requis en terme de configuration matérielle (mémoire, CPU…)
- Vérifier si des programmes additionnels requis sont installés (ex : .Net Framework)
- Se renseigner sur l'éditeur, s'il est de confiance ou pas
- Se renseigner sur un éventuel suivi technique à long-terme (LTS : Long-Term Support)
- Se renseigner sur les avis dans Google ou les sites spécialisés…

<div className="seperator" />
## Pendant l'installation
- Bien lire toutes les alertes du système d'exploitation lors de l'installation
- Annuler l'installation si l'anti-virus s'y oppose
- Bien faire attention à redémarrer si l'assistant d'installation le demande
- Ne pas installer deux programmes en même temps. Risque de conflits

<div className="seperator" />
## Après l'installation
- Rechercher des mises à jour
- Tester le fonctionnement

<div className="seperator" />
## Supprimer et Réparer

- Il peut y avoir, dans le menu "installer/désinstaller un programme", un agent de désinstallation
pour un logiciel en particulier (c'est souvent le cas des antivirus). Cet outil de désinstallation
peut aussi contenir un outil de réparation du logiciel.
- Dans le menu "installer/désinstaller un programme", en cliquant-droit sur un programme
installé
