# Raccourcis Clavier Windows

## Raccourcis Clavier Généraux

| Touche(s)                        | Action                                                                 |
|----------------------------------|-----------------------------------------------------------------------|
| `F1`                             | Afficher l’aide                                                    |
| `Ctrl + C` ou `Ctrl + Inser`     | Copier l’élément sélectionné                                        |
| `Ctrl + X`                       | Couper l’élément sélectionné                                        |
| `Ctrl + V` ou `Maj + Inser`      | Coller l’élément sélectionné                                        |
| `Ctrl + Z`                       | Annuler une action                                                 |
| `Ctrl + Y`                       | Restaurer les modifications                                        |
| `Suppr` ou `Ctrl + D`            | Supprimer l’élément sélectionné et le déplacer vers la Corbeille   |
| `Maj + Suppr`                    | Supprimer l’élément sans passer par la Corbeille                   |
| `F2`                             | Renommer l’élément sélectionné                                     |
| `Ctrl + Flèche Droite`           | Avancer le curseur au début du mot suivant                         |
| `Ctrl + Flèche Gauche`           | Avancer le curseur au début du mot précédent                       |
| `Ctrl + Flèche Bas`              | Avancer le curseur au début du paragraphe suivant                  |
| `Ctrl + Flèche Haut`             | Avancer le curseur au début du paragraphe précédent                |
| `Ctrl + A`                       | Sélectionner tous les éléments dans un document ou une fenêtre     |
| `Alt + Tab`                      | Passer d’un élément ouvert à un autre                              |
| `Alt + F4`                       | Fermer l’élément actif ou quitter le programme                    |
| `Ctrl + Maj + Échap`             | Ouvrir le Gestionnaire des tâches                                 |

## Raccourcis Clavier de la Touche Windows

| Touche(s)                     | Action                                                                 |
|-------------------------------|-----------------------------------------------------------------------|
| `Win`                         | Ouvrir ou fermer le menu Démarrer                                   |
| `Win + D`                     | Afficher le bureau                                                  |
| `Win + M`                     | Réduire toutes les fenêtres                                         |
| `Win + Maj + M`               | Restaurer les fenêtres réduites                                     |
| `Win + E`                     | Ouvrir l’Explorateur Windows                                        |
| `Win + L`                     | Verrouiller l’ordinateur ou changer d’utilisateur                  |
| `Win + R`                     | Ouvrir la boîte de dialogue Exécuter                               |
| `Win + T`                     | Parcourir les programmes sur la barre des tâches                   |
| `Win + Flèche Haut`           | Agrandir la fenêtre                                                |
| `Win + Flèche Bas`            | Réduire la fenêtre                                                 |
| `Win + Flèche Gauche`         | Agrandir la fenêtre sur le côté gauche                             |
| `Win + Flèche Droite`         | Agrandir la fenêtre sur le côté droit                              |
| `Win + P`                     | Choisir un mode d’affichage de présentation                        |

---

## Raccourcis dans l’Explorateur Windows

| Touche(s)                      | Action                                                                 |
|--------------------------------|-----------------------------------------------------------------------|
| `Ctrl + N`                     | Ouvrir une nouvelle fenêtre                                          |
| `Ctrl + W`                     | Fermer la fenêtre actuelle                                          |
| `Ctrl + Maj + N`               | Créer un dossier                                                   |
| `Alt + P`                      | Afficher le volet de visualisation                                 |
| `Alt + Flèche Gauche`          | Afficher le dossier précédent                                      |
| `Alt + Flèche Haut`            | Afficher le dossier parent                                         |
| `Ctrl + Roulette de Souris`    | Modifier la taille du texte, des fichiers ou des icônes            |
| `Alt + Entrée`                 | Ouvrir les propriétés de l’élément sélectionné                     |

---

## Raccourcis Clavier dans les Boîtes de Dialogue

| Touche(s)                      | Action                                                                 |
|--------------------------------|-----------------------------------------------------------------------|
| `Ctrl + Tab`                   | Avancer dans les onglets                                             |
| `Ctrl + Maj + Tab`             | Reculer dans les onglets                                             |
| `Tab`                          | Avancer dans les options                                             |
| `Maj + Tab`                    | Reculer dans les options                                             |
| `Alt + Lettre Soulignée`       | Exécuter une commande ou sélectionner une option                    |
| `Entrée`                       | Valider une commande                                                |
| `Barre d’Espace`               | Activer ou désactiver une case à cocher                              |

---

## Raccourcis Clavier de la Barre des Tâches

| Touche(s)                      | Action                                                                 |
|--------------------------------|-----------------------------------------------------------------------|
| `Maj + Clic`                   | Ouvrir une nouvelle instance d’un programme                         |
| `Ctrl + Maj + Clic`            | Ouvrir un programme en tant qu’administrateur                      |
| `Ctrl + Clic`                  | Parcourir les fenêtres d’un groupe                                  |
