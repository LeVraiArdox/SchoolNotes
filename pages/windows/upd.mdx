# Les mises à jour Windows

Une mise a jour est un ensemble de modifications apportées a un logiciel pour corriger des bugs (donc un correctif) ou pour, 
améliorer des fonctionnalités ou ajouter de nouvelles fonctionnalités (donc une amélioration). 
<br />
<div class="p-4 font-mono border-l-2 border-primary bg-mbg my-6">
Concernant les mises à jour Windows, on parle de mises à jour cumulatives, et elles se présentent
sous la forme KB123456 (KB = Knowledge Bulletin).
</div>

# La mise à niveau
La mise à niveau fait passer à la **version supérieure du logiciel**. Elle apporte des modifications
**importantes** du logiciel (nouvelles fonctionnalités, nouvelle interface, adaptation à de nouvelles
architectures…). En règle générale, cela est payant mais certaines mises à niveau peuvent être
gratuites.
<br />
Une mise à niveau est aussi matérielle.
<br />
*Exemple : numéro de version passe de 1.0 à 2.0*
*Exemple : passage de d'un CPU Intel Core i5 à un Intel Core i7*

# Le flashage
Le flashage est une mise à jour du **micrologiciel** (firmware) d’un appareil. Il est souvent utilisé pour
mettre à jour le BIOS d’un ordinateur, le micrologiciel d’un routeur, d’un disque dur, d’un appareil
photo, d’un smartphone, etc.
<br />
Le flashage est souvent nécessaire pour corriger des bugs, améliorer la compatibilité avec de nouveaux
matériels, ou ajouter de nouvelles fonctionnalités.
<div class="p-4 font-mono border-l-2 border-red bg-mbg my-6">
Contrairement aux autres types de mises à jour, celle-ci doit se dérouler correctement et sans
interruption. Dans le cas où un problème survient durant un flashage, l’appareil pourrait
irrémédiablement devenir totalement inutilisable.
</div>

# Le retour en arrière (downgrade)
Le retour en arrière est une opération qui consiste à revenir à une version antérieure d’un logiciel.
Cette opération est souvent nécessaire lorsqu’une mise à jour a provoqué des dysfonctionnements ou
des incompatibilités. 
<br />
Le retour en arrière est souvent possible, mais il peut être difficile voire impossible dans certains cas.
Une fin de support ou une **obsolescence programmée** peut être à l’origine d’une décision d’un retour
à une version précédente.

# La configuration d'une (Reset ou Wipe)

La configuration d'une machine consiste à remettre à zéro les paramètres de celle-ci.
Cela peut être utile pour résoudre des problèmes de performances, de bugs, ou pour effacer des données
personnelles avant de vendre ou de donner l’appareil. 
- le **Soft reset** est une réinitialisation logicielle qui ne supprime pas les données personnelles.
- le **Hard reset (ou wipe)** est une réinitialisation matérielle qui supprime toutes les données personnelles.
